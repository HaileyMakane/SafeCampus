# SafeCampus
SafeCampus is an AI-powered emergency response ecosystem designed to enhance campus safety.

Features:
• Wearable SOS bracelet
• Automatic distress detection
• Clip-on emergency camera
• Firebase cloud synchronization
• AI threat prioritization
• Security dashboard
• Telegram patrol bot
• Student mobile application

Technologies:
ESP32 | Firebase | YOLOv8 | CNN | Streamlit | Kivy | Python
